Executive Summary

SmartPDF is a production-ready Flutter application that enables users to:

  • Authenticate via email/password, Google Sign-In, or as a guest (no account required)
  • Upload & Store PDF files securely in Firebase Cloud Storage
  • Convert between PDF and Word document formats
  • Manage PDFs with search, share, and download capabilities
  • Access across iOS, Android, and Web platforms

The app implements clean architecture using GetX for state management, Firebase for backend services, and Material Design 3 for responsive UI. All core features are implemented and tested.

Quick Start Guide

Installation

git clone <repository-url>
cd smartpdf
flutter pub get
flutter run -d <device>

Test Features

  1. Click "Continue as Guest" to access without login
  2. Try "Email/Password" to create and login with account
  3. Use "Google Sign-In" button (mobile only)
  4. Upload a PDF file (mobile only)
  5. Convert PDF to Word or Word to PDF
  6. View conversion history and manage files
  7. Edit user profile and switch themes

Setup Requirements

Important: Before running on mobile devices, configure Firebase:
  1. Create Firebase project at https://console.firebase.google.com
  2. Enable Authentication (Email/Password, Google)
  3. Enable Cloud Firestore and Cloud Storage
  4. Download google-services.json (Android) and GoogleService-Info.plist (iOS)
  5. Place these files in the correct directories (already done in this repo)

Architecture Overview

SmartPDF follows Clean Architecture with clear separation of concerns:

  • Services Layer: Authentication, PDF, Storage, Conversion services
  • Controllers Layer: Business logic with GetX reactive state
  • Views Layer: UI screens with Material Design 3
  • Routes: Navigation using GetX routing
  • Themes: Centralized styling and Material Design 3

Troubleshooting

Common Issues

Issue: App crashes on startup
Solution: Run `flutter clean && flutter pub get`, then `flutter run`
Issue: Firebase errors (auth, storage)
Solution: Verify google-services.json and GoogleService-Info.plist are present and match your Firebase project
Issue: PDF conversion not working
Solution: PDF conversion only works on iOS/Android, not on Web
